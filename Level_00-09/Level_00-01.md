## 🎯 Level 0 -> Level 1
**Objective:** The password for the next level is stored in a file called `readme` located in the home directory.

**Concepts Required:** Basic file reading.
**Commands Used:** `ls`, `cat`

**Solution:**
```bash
bandit0@bandit:~$ ls
readme
bandit0@bandit:~$ cat readme
[REDACTED_PASSWORD]
```
